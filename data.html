MSc Data Science

Analyze data and build intelligent systems

Course Overview

Data Science focuses on analyzing large data sets to discover patterns and support decision making. It helps turn raw data into useful information.

Students learn how to collect, organize, and analyze data using tools and techniques. These skills are useful in many fields, helping organizations make better and smarter decisions.

Main Subjects

  • Data Analysis
  • Machine Learning
  • Statistics
  • Big Data Technologies

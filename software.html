BSc Software Engineering

Design and develop software systems

Course Overview

Software Engineering focuses on designing, developing, and maintaining software applications. It teaches students how to build reliable, efficient, and user-friendly software that meets real-world needs.

Students learn programming, software design principles, testing, and project management. These skills prepare them for careers in software development, application design, and technology innovation.

Main Subjects

  • Software Design
  • System Development
  • Database Systems
  • Software Testing

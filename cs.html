BSc Computer Science

Learn programming and software development

Course Overview

Computer Science teaches programming, algorithms, and software development used in modern technology. It helps students understand how computers work and how to build software and applications.

Students learn to solve problems, write code, and create systems used in websites, apps, and other digital tools. These skills are important for many careers in today’s technology-driven world.

Main Subjects

  • Programming Fundamentals
  • Data Structures
  • Operating Systems
  • Artificial Intelligence
